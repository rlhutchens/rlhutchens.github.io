Silent File Scanner
CLI Tool  ·  Python

A cross-platform command-line tool for finding and removing silent audio files from DAW session archives. Built out of a direct need to free up disk space consumed by silent takes and unused inputs from recording projects. The tool recursively scans directories, identifies silent audio files based on configurable thresholds, generates structured JSON reports grouped by session, and routes flagged files to the OS Recycle Bin.

Python · Click · NumPy · soundfile · pytest Source
Frequency to Cents Converter
Web App  ·  Vue + TypeScript  

Converts lists of frequency values in hertz to cent offsets relative to root notes. Useful for musicians interested in working with discrete frequencies in a non-standard tuning system. I consider it a companion tool to composing scales in Scala.
